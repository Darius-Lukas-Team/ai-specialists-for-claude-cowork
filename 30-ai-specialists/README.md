# AI Specialists for Claude Cowork

30 specialist AI assistants for business growth — covering social media, copywriting, sales, marketing, customer support, and more. Each helper is a Claude Cowork skill with its own persona, expertise, frameworks, and response structure.

## Install

1. Open the Claude app
2. Go to **Cowork** > **Plugins** > **+** > **Upload**
3. Upload the plugin zip file
4. You're all set!

## Usage

Invoke any helper by typing `/skill-name` in chat, or let Claude use them automatically for relevant tasks.

Use `/ai-helper` as a smart router — describe what you need and it will pick the right specialist(s) automatically.
